# Настройка

Чтобы установить и запустить бота, нужно сделать несколько простых шагов.

# Установка

git clone https://github.com/sashafromlibertalia/sparkle.git
1

# Восстановление зависимостей

npm install
1

# Токен

WARNING

Бот использует Longpoll. О том, как его настроить, можно почитать тут (opens new window)

Чтобы бот работал с вашим сообществом, нужно вставить полученный токен и ID группы, от имени которой будет работать бот. Для этого в файле .env нужно добавить свежий ключ:

TOKEN=YOUR_GENERATED_TOKEN
GROUP_ID=YOUR_GROUP_ID
1
2

После этого бот внутри вашей группы будет работать.
